# Share Your Workflow

Follow these steps to share your workflow with the KadangKesel community:

## 1. Export Your Workflow

* Open **StockStudio**
* Select the workflow you want to share
* Export the workflow file (usually in JSON format)

## 2. Create Description File (.md)

* Create a Markdown file (`.md`) for your workflow
* Use the provided template:

```md
# [Workflow Title]

[Short description of what your workflow does]

## Tags

tag1, tag2, tag3

## Author

Your Name or Community
```

* Make sure the description is clear and easy to understand

## 3. Prepare Your Files

Your submission should include:

* Workflow file (e.g. `.json`)
* Description file (`.md`)

Example:

```text
your-workflow.json
your-workflow.md
```

## 4. Upload to GitHub

* Fork the repository
* Add your files to the appropriate folder
* Commit your changes with a clear message

Example:

```
docs: add workflow for adobe stock metadata generation
```

## 5. Create a Pull Request

* Open a Pull Request (PR) to the main repository
* Add a short explanation of your workflow
* Wait for review and approval

## Tips

* Keep your workflow useful and reusable
* Use clear naming for files
* Avoid sensitive or private data
* Make sure your workflow works before submitting
